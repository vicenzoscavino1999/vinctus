# Manual regression checklist

Use this for every block/PR. Keep results short and explicit.

## Steps and expected results

- Login/auth: user can sign in and session persists after refresh.
- Main navigation: top-level routes load and do not crash.
- Feed: open feed and scroll one screen without errors.
- Primary CRUD: create, edit, delete one core entity.
- Upload/download: upload and view/download a file (if applicable).
- Payments: complete a basic payment flow (if applicable).
